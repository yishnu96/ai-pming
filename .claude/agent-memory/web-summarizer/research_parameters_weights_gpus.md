## Key Research Findings

### Parameters & Weights
- Parameters = internal numbers AI learns during training
- Weights = numerical values measuring connection strength between nodes
- Learning happens by adjusting weights based on feedback
- Single weight changes can have exponential impact on output

### Model Size Reality (2026 Critical Finding)
- **Bigger ≠ Better**: 7B fine-tuned models often outperform 70B general models
- 7B models: Fast, cost-effective, domain-specific advantage
- 70B models: Slower, expensive, better for general tasks
- Enterprise lesson: Domain depth beats scale breadth

### GPUs Essentials
- AI = billions of parallel matrix multiplications
- CPU = sequential (too slow for AI)
- GPU = parallel architecture (thousands of cores)
- GPU memory is the bottleneck
- GPU availability drives AI infrastructure costs
